Se adjunta codigo inicial trabajado, incluye la separacion del codigo usando
el patron de diseño MVC. Para el modelo usamos la libreria pydicom que es para extraer los datos
de los archivos y para la vista se usa tkinter.
Actualmente se puede seleccionar una imagen tipo dicom y ésta extraerá la fecha.
Para ejecutar el programa, se debe ejecutar el archivo main.py. 
python main.py



Librerias a instalar previamente:
(pip install <nombre>)
Numpy
Pillow
pydicom
